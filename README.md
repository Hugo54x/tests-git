# Performance Tests for Git

## Getting Started

### Pre-requsites

- Visual Studio Code

- Remote Container Extension

### Installation

Run `CTRL+SHIFT+P` and `Remote Container: Open folder in container`.

Your environment will be automatically configured.

### Usage

Run `python3 git_branch_perf.py`
